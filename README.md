# Pico8 Games
### Español
Una recopilación de juegos desarrollados con Lua en el motor Pico8.

_Todos los juegos presentes en este repositorio son proyectos personales, con el mero objetivo de aprender sobre el desarrollo de videojuegos a un nivel fundamental, siendo única y exclusivamentes de caracter experimental_

#### Instrucciones de ejecución

1. Descargue el juego de su elección.
2. Diríjase a la página oficial de [Pico8 Edu](https://www.pico-8-edu.com/).
3. Pulse en el icono de play en pantalla para inciar la consola virtual.
4. Arrastre el archivo del juego a la página para cargarlo en memoria.
5. Pulse **Ctrl + R** para ejecutarlo (y para reiniciarlo).

También puede pulsar la tecla **Esc** una vez (o dos dependiendo en que pantalla se encuentre) para ver el código, sprites, mapas, etc.

---
### English
A compilation of games made in Lua and created using Pico8 as a game engine

_All the games present in this repository are personal projects, made with the goal of learning about game development at a fundamental level, and therefore being of an experimental nature_

#### Execution steps

1. Download the desired game.
2. Navigate to the official [Pico8 Edu](https://www.pico-8-edu.com/) page.
3. Click the "play" icon onscreen to start the virtual console / terminal.
4. Drag the game's file to the webpage to load it into memory.
5. Press **Ctrl + R** to run it (and to restart it).

You can also press the **Esc** key one time (or two depending on what screen you're currently at) to view the code, sprites, maps, etc.
