# PICO-8 Games
### Español
Una recopilación de juegos desarrollados con Lua utilizando el motor PICO-8.

_Todos los juegos presentes en este repositorio son proyectos personales, con el mero objetivo de aprender sobre el desarrollo de videojuegos a un nivel fundamental, siendo única y exclusivamentes de carácter experimental_

#### Instrucciones de ejecución

1. Descargue el juego de su elección.
2. Diríjase a la página oficial de [PICO-8 EDU](https://www.pico-8-edu.com/).
3. Pulse en el icono de "Play" en pantalla para inciar la consola virtual.
4. Arrastre el archivo del juego a la página para cargarlo en memoria.
5. Pulse **Ctrl + R** para ejecutarlo (y para reiniciarlo).

También puede pulsar la tecla **Esc** una vez (o dos, dependiendo de la pantalla en la que se encuentre) para ver el código, sprites, mapas, etc.

### English
A compilation of games developed in Lua using PICO-8 as the game engine

_All the games present in this repository are personal projects, created with the goal of learning game development at a fundamental level, and therefore experimental in nature_

#### Execution steps

1. Download the desired game.
2. Navigate to the official [PICO-8 EDU](https://www.pico-8-edu.com/) page.
3. Click the "Play" icon on screen to start the virtual console.
4. Drag the game's file to the webpage to load it into memory.
5. Press **Ctrl + R** to run it (and to restart it).

You can also press the **Esc** key once (or twice, depending on the current screen) to view the code, sprites, maps, etc.
